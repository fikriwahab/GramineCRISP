vault file: metadata file
Misal suatu aplikasi buka 3 file, masing-masing akan di enkripsi hash, informasi ini ada di dalam vault file tersebut (bisa ratusan filenya, tapi vault cuma 1), dia akan kasih info data/skema tentang file yang sudah dihash dan enkripsi ini

enclave: memori ada ketika suatu aplikasi running (bukan file), jadi aman dari intervensi luar

vault tidak wajib kalau kita mau ngoprek protected file, ditaro di metadata emg memberikan fleksibilitas (untuk future work), kalau ribet ga masalah digabung jadi satu. embed mono counter disitu ga masalah

Pseudocode syscall:
int main(){
    printf("hai hai fasilkom");
    
    int fd =fopen("a.txt", w);
    fwrite(buf, len(buf), ?, fd);
    fsync(fd)
    close(fd)
}
